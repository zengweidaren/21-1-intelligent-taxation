# 智能财务部 - 3D虚拟办公室

基于 Three.js 构建的3D虚拟办公室，支持360°旋转查看。

## 功能

- 🖱️ 拖拽旋转视角
- 🔍 滚轮缩放
- 👆 点击工位查看人员信息
- 🖼️ 文化墙展示（12个月 + 表彰墙）

## 部署到 Gitee Pages

1. 将整个文件夹上传到 Gitee 仓库
2. 进入仓库 → 设置 → Gitee Pages
3. 选择分支（master/main）和目录（/）
4. 点击部署
5. 访问 `https://你的用户名.gitee.io/智能财务部/`

## 本地预览

直接用浏览器打开 `index.html` 即可。

## 技术栈

- Three.js 3D渲染
- OrbitControls 视角控制
- 原生JavaScript
